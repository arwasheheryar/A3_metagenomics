# Assignment 3: Shotgun Metagenomics
Taxonomic classification and diversity analysis of gut microbiomes from vegan and omnivore subjects.
Data: De Filippis et al. 2019 (SRP126540).

## Structure
- pipeline.sh — bash pipeline (fastp, Kraken2, Bracken, kraken-biom)
- analysis.R  — R analysis (phyloseq, alpha/beta diversity, LinDA)
- results/    — BIOM table and figures
